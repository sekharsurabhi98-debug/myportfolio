🔥 Tech Glossary

HTML
HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the web. A web browser reads an HTML document, interprets the tags and attributes, and renders the content as a multimedia web page
CSS
CSS (Cascading Style Sheets) is a stylesheet language used to style and format a web page written in a markup language like HTML. It defines how HTML elements are displayed on different media, controlling the layout, colors, fonts, and overall visual appearance of a website
JS
JavaScript (JS) is a high-level, interpreted programming language that adds dynamic and interactive functionality to websites. As one of the core technologies of the World Wide Web, it works alongside HTML (content structure) and CSS (styling) to create a complete and engaging user experience
